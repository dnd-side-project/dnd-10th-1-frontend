### 🚀 로컬환경에서 실행하는 방법

**(필수) node_modules 설치**
> 호환되지 않는 라이브러리가 있기 때문에 --force 명령어로 설치하여야 합니다.

```shell
npm install --force
```

**개발 환경에서 확인하는 방법**
```shell
npm run dev
```

**사용자 환경에서 확인하는 방법**
```shell
npm run build

npm run start
```

**StoryBook 확인하는 방법**
```shell
npm run storybook
```
